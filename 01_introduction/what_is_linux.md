# What is Linux?

## Description
Linux is an open-source operating system used in servers,
cloud platforms, development environments, and embedded systems.
Created by Linus Torvalds in 1991.

## Key Features
- Free and open source
- Secure and stable
- Highly customizable
- Widely used in IT industry

## Where Linux is Used
- Web servers
- Cloud platforms (AWS, Azure)
- Android devices
- DevOps and cybersecurity.
