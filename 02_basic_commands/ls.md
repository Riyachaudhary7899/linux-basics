## 1. ls Command (List Files)

### Description
The `ls` command is used to list files and directories.

### Syntax
```bash
ls

###Common Options
ls -l    # long listing (permissions, size, date)
ls -a    # show hidden files
ls -lh   # human-readable file sizes
